# Giffer

## English

### Short Description
Giffer is a desktop and web tool for turning image sequences or a single video into GIF or MOV exports with live preview, framing, cropping, and transform controls.

### Features
- Import multiple images at once.
- Import a single video and convert it into frames.
- Export as GIF.
- Export as MOV.
- Real-time preview while editing.
- Images to GIF mode.
- Video to GIF mode.
- Adjustable frame delay in milliseconds.
- Adjustable frame step / frame skipping.
- Separate MOV FPS control.
- Stack mode for PNG sequences.
- Automatic stack disable when JPG/JPEG files are present.
- Left/right split preview in video mode.
- Original video playback controls in video mode.
- Video timeline scrubbing in video mode.
- Video trim range controls in video mode.
- GIF preview autoplay.
- GIF preview pause while original video plays for better performance.
- Hover preview on frame thumbnails.
- Remove individual frames from the list.
- Export visible video frames as ZIP.
- Output size mode: long edge.
- Output size mode: original size.
- Custom long-edge output value.
- GIF optimization percentage control.
- Transparent background option.
- Custom preview and export background color.
- Frame presets: no frame, 1:1, 16:9, 9:16.
- Crop area editing.
- Transform mode for moving content inside the frame.
- Scale slider for zooming content.
- Show outside / hide outside frame option.
- Frame drag and resize controls.
- Original-size export based on the largest loaded frame.
- Desktop save dialog for exports.
- Web download flow for exports.
- Release packaging for web, Windows, and Itch builds.
