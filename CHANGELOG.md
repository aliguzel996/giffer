# Changelog

This changelog tracks public-facing project changes for Giffer.

## Unreleased

- Reserved for the next release.

## 0.1.0 - 2026-05-10

- Added hybrid web and Windows packaging flow.
- Added image-sequence and single-video import modes.
- Added GIF and MOV export.
- Added crop, frame presets, transform controls, and output sizing.
- Added frame reordering, mirror loop for image mode, and ZIP export for visible video frames.
- Added machine-readable project metadata files.
